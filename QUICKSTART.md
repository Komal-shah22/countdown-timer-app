# 🚀 Quick Start Guide

## View Your Professional Timer

### Option 1: Development Mode
```bash
npm run dev
```
Then open: http://localhost:3000

### Option 2: Production Build
```bash
npm run build
npm start
```

## 🎯 What to Test

### 1. **Start Page**
- Click "Get Started" button
- Notice the floating clock icon animation
- Check the feature cards at the bottom

### 2. **Input Page**
- Try the quick preset buttons (5 Min, 30 Min, etc.)
- Or select a custom date/time
- Notice validation for past dates

### 3. **Timer Display**
- Watch the circular progress ring
- See the animated timer cards
- Toggle sound notifications (speaker icon)
- Try Pause/Resume/Reset buttons

### 4. **Dark Mode**
- Click the sun/moon icon in top-right
- Watch smooth color transitions
- Theme persists on page reload

### 5. **Completion**
- Set a 10-second timer to see:
  - Confetti animation
  - Sound notification (if enabled)
  - Celebration screen
